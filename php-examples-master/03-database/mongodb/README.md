Install mongodb:
```
$ sudo apt-get install mongodb

$ mongo
```

Install PHP module:

https://docs.mongodb.org/ecosystem/drivers/php/

```
$ sudo apt-get install php5-dev php5-cli php-pear
$ sudo pecl install mongo
To upgrade mongo
$ pecl upgrade mongo

Open your php.ini file and add to it:

extension=mongo.so
```
